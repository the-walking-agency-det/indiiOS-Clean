# Publicist Module (RC1)

The Publicist module automates the high-value task of media outreach, press release generation, and crisis management. It serves as a digital **PR Agent**, ensuring the artist's story is told correctly across the media landscape.

## 📰 Key Features
- **Press Release Generator:** Creates professional, AP-style press releases from track metadata and creative briefs.
- **Media List Manager:** Specialized contact database for music blogs, radio stations, and influencers.
- **Outreach Orchestrator:** Automated email drafting and follow-up scheduling.
- **Crisis Response:** Rapid AI-driven drafting of official statements for sensitive situations.
- **EPK Distribution:** Direct delivery of Electronic Press Kits to media contacts.

## 🏗️ Technical Architecture
- **`PublicistTools`**: A specialized set of agent tools for PR-specific reasoning and writing.
- **Markdown Templates:** Standardized formats for press materials.
- **Marketing Integration:** Synchronizes PR activities with broader release campaigns.
