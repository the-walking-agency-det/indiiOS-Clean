## 1. Description

The utility file `src/utils/async.ts` provides critical workflow functions (`delay`, `retry`) but currently lacks unit tests. This proposal introduces a comprehensive test suite using Vitest to ensure reliability and prevent regressions in these fundamental helpers.

## 2. Technical Scope

- Create `src/utils/async.test.ts`.
- Coverage for `delay()`: Verify it waits approximately the correct time.
- Coverage for `retry()`:
  - Successful first execution.
  - Successful execution after 1 failure.
  - Final failure after all retries are exhausted.
  - Exponential backoff interval calculation verification.
