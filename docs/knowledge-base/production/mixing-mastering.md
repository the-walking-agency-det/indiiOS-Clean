# Mixing & Mastering Guide

## Overview

Mixing and mastering are the final creative and technical stages that transform raw recordings into polished, release-ready music. This guide explains what to expect, file requirements, revision policies, and how to work effectively with mix and mastering engineers.

---

## Mixing: What It Is

Mixing is the process of balancing, processing, and blending individual tracks (stems) into a cohesive stereo or multichannel presentation. A mix engineer combines artistic judgment with technical expertise to create clarity, depth, and emotion.

### What Mixing Includes

| Task | Description |
|------|-------------|
| **Gain Staging** | Setting optimal levels for each track |
| **EQ** | Shaping frequency balance for clarity and separation |
| **Compression** | Controlling dynamics and adding punch/glue |
| **Panning** | Placing elements in the stereo field |
| **Effects** | Reverb, delay, modulation, saturation |
| **Automation** | Dynamic changes in level, effects, and panning over time |
| **Editing** | Cleaning up noises, fades, timing issues |
| **Vocal Processing** | Tuning, de-essing, special effects |

### Mixing Formats

- **Stereo Mix:** Standard 2-channel mix for most releases
- **Instrumental:** Mix without lead vocals
- **TV Track:** Mix without lead vocals but with background vocals
- **A Cappella:** Vocals only
- **Stems:** Subgroups (drums, bass, guitars, vocals, etc.) for further processing

## Preparing Files for Mixing

### File Specifications

| Parameter | Recommended | Acceptable |
|-----------|-------------|------------|
| **Format** | WAV or AIFF | FLAC (lossless) |
| **Bit Depth** | 24-bit | 32-bit float |
| **Sample Rate** | 48kHz or 96kHz | 44.1kHz |
| **Channels** | Mono (individual tracks) | Stereo (only for stereo sources) |
| **Headroom** | -6dB to -12dB peak | Never clip (0dB+) |

### Track Preparation Checklist

- [ ] **Consolidate/Export** all tracks to start at 00:00:00
- [ ] **Label clearly:** `01_Kick.wav`, `02_Snare.wav`, `03_BassDI.wav`
- [ ] **Remove plugins** (unless specific sound is desired)
- [ ] **Include notes:** Tempo, key, reference tracks, special requests
- [ ] **Organize by type:** Drums, Bass, Guitars, Keys, Vocals, Effects
- [ ] **Include MIDI** if virtual instruments need replacing
- [ ] **Note tuning:** If tracks were tuned, mention software used

### Reference Tracks

Provide 2–3 commercial releases that represent your sonic goals:
- Similar genre and instrumentation
- Specific elements you like ("the vocal treatment on Track A")
- What you DON'T want ("avoid the harshness in Track B's chorus")

### Mix Notes Template

```
SONG: [Title]
ARTIST: [Name]
BPM: [Tempo]
KEY: [Key]

REFERENCE TRACKS:
1. [Artist - Song] - [What you like]
2. [Artist - Song] - [What you like]

MIX NOTES:
- Lead vocal should be [forward/warm/intimate]
- Drums should feel [punchy/roomy/tight]
- Bass should [drive the groove/sit back]
- Guitar solo at 2:15 should [cut through/blend]

SPECIAL REQUESTS:
- Keep the vocal ad-lib at 1:45 prominent
- Add telephone effect on bridge vocals
- Create space for the synth that enters at 0:45

TECHNICAL NOTES:
- Vocal was tuned with Melodyne
- Guitar amp sim used: [Name]
- Virtual drums: [Name/Version]
```

## The Mixing Process

### Step 1: Organization (Engineer)
- Import and organize tracks
- Color-code by instrument type
- Create bus/aux tracks for routing
- Set up monitoring and reference tracks

### Step 2: Rough Balance (30–60 min)
- Quick fader balance to hear the song
- Identify problem areas
- Establish overall energy and flow

### Step 3: Detailed Mixing (4–8 hours)
- Process individual tracks (EQ, compression)
- Create spatial depth (reverb, delay)
- Automation passes for dynamics
- Vocal focus and treatment

### Step 4: Review & Revisions
- Engineer sends reference bounce
- Artist provides feedback
- Revisions implemented
- Final approval

### Step 5: Delivery
- Export final mix(es)
- Create stems if requested
- Deliver files via download link
- Archive session

## Mix Revisions

### How to Give Good Feedback

**DO:**
- Reference timestamps ("at 1:23 in the second verse")
- Use descriptive language ("more presence," "less muddy")
- Compare to reference tracks ("like the vocal in Track A")
- Prioritize changes ("most important: vocal clarity")

**DON'T:**
- Give technical instructions ("add 2kHz boost")—trust the engineer
- Request changes that contradict your references
- Send feedback immediately—listen on multiple systems first
- Ask for unlimited revisions (respect the process)

### Common Revision Requests

| Issue | Typical Fix |
|-------|-------------|
| Vocal too quiet/ loud | Level automation |
| Bass not audible | EQ carve, compression |
| Drums lack punch | Transient shaping, parallel compression |
| Harshness in mix | EQ cuts, de-essing, saturation |
| Lacks depth | Reverb adjustments, spatial processing |
| Muddy low end | High-pass filtering, bass management |

### Revision Rounds

Most engineers include 2–3 revision rounds:
1. **Round 1:** Major issues (balance, overall tone)
2. **Round 2:** Fine-tuning (specific moments, effects)
3. **Round 3:** Final polish (minor tweaks)

Additional rounds typically cost $50–$150 each.

## Mastering: What It Is

Mastering is the final step before distribution. A mastering engineer optimizes the mix for playback across all systems and formats, ensuring consistency and competitive loudness.

### What Mastering Includes

| Task | Purpose |
|------|---------|
| **EQ** | Final tonal balance adjustments |
| **Compression** | Glue and cohesion |
| **Limiting** | Achieving target loudness |
| **Stereo Enhancement** | Width and imaging optimization |
| **Sequencing** | Album flow and spacing |
| **Format Optimization** | Specific masters for streaming, vinyl, CD |
| **Quality Control** | Final error checking |

### Mastering Formats

- **Streaming Master:** Optimized for -14 to -16 LUFS
- **CD Master:** -9 to -11 LUFS, Red Book standard
- **Vinyl Master:** Specific limitations (sides < 20 min, bass management)
- **Instrumental/TV Track Masters:** Matching lead vocal master

## Preparing for Mastering

### File Requirements

| Parameter | Specification |
|-----------|---------------|
| **Format** | WAV or AIFF |
| **Bit Depth** | 24-bit or 32-bit float |
| **Sample Rate** | Original session rate (don't convert) |
| **Headroom** | -3dB to -6dB peak (leave room for processing) |
| **Processing** | No limiting, minimal compression on mix bus |

### Mix Bus Guidelines

**DO:**
- Leave headroom (-3dB to -6dB)
- Remove mix bus limiting
- Export at original sample rate
- Include fades at beginning/end

**DON'T:**
- Send clipped files (peak above 0dB)
- Apply heavy mix bus compression
- Use "maximizer" or "loudness" plugins
- Convert sample rates (44.1kHz → 48kHz)

### Reference Information

Provide the mastering engineer:
- ISRC codes (if available)
- Track order (for albums/EPs)
- Crossfade preferences between tracks
- Target loudness (if specific)
- Streaming platforms prioritized

## Working with Engineers

### Finding Mix/Mastering Engineers

**Platforms:**
- **SoundBetter:** soundbetter.com — Vetted professionals with reviews
- **AirGigs:** airgigs.com — Mixing and mastering services
- **Fiverr/Upwork:** Budget options, vet carefully
- **Gearslutz/GroupDIY:** Forums with engineer recommendations

**Direct Contact:**
- Check credits on releases you admire
- Contact via artist/label websites
- Attend audio conferences (AES, NAMM)

### Evaluating Engineers

**Listen to their portfolio:**
- Do they work in your genre?
- Consistent quality across different artists?
- Can you hear the "before" to judge improvement?

**Check reviews/testimonials:**
- Communication quality
- Turnaround reliability
- Revision flexibility

**Questions to Ask:**
- What's your typical revision policy?
- What file formats do you deliver?
- Do you offer stem exports?
- What's your archive/backup policy?

## Common Mistakes

### During Recording
- **Over-processing on the way in**—hard to fix later
- **Not recording enough headroom**—clipping ruins takes
- **Poor room acoustics**—can't EQ out bad room sound

### During Mixing Prep
- **Sending MP3s**—lossy compression destroys quality
- **Poorly labeled files**—wastes engineer time
- **Too many plugins on tracks**—limits engineer flexibility

### During Mastering Prep
- **Over-limiting the mix**—no headroom for mastering
- **Multiple sample rate conversions**—degrades quality
- **Ignoring format requirements**—vinyl needs different treatment

### Communication
- **Vague feedback**—"make it sound better" isn't actionable
- **Too many cooks**—one clear voice for decisions
- **Rushing revisions**—sleep on it before requesting changes

## DIY vs. Professional

### When to DIY Mixing
- Learning and experimentation
- Demos and pre-production
- Electronic music with precise control needs
- Tight budget with time to learn

### When to Hire a Professional
- Final releases for distribution
- Complex productions (live instruments, many tracks)
- Genre expertise needed
- Objective ears required

### When to DIY Mastering
- Quick references and roughs
- Learning the craft
- Very controlled listening environment available

### When to Hire a Mastering Engineer
- Final release masters
- Album sequencing and cohesion
- Multiple format requirements
- Competitive loudness needed
